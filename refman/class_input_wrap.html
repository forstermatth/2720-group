Set passed vars based on the node name passed.

Helper function to reduce duplicate code and make it more readable. Only one var is set per call, based on the name of the node. Used right after parsing a node to set the corresponding variable to prepare for constructing the object.

Parameters
nodethe name of the node, dictates what var is set
valuethe value of the node that the var will be set to.
_namethe course/lab name variable to set
_startthe course/lab start time variable to set
_endthe course/lab end time variable to set
_daysthe course/lab days variable to set
_locthe course/lab location variable to set
_idthe course/lab id variable to set
